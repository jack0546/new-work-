# **App Name**: Elegance Boutique

## Core Features:

- User Authentication & Profiles: Allow users to register, log in, manage their profiles, and reset passwords using Firebase Authentication.
- Product Catalog Browsing: Display luxury fashion products categorized by type (handbags, high heels), with ability to view detailed product information, multiple images, and zoom effects. Basic filtering by category and sorting by price/newest are included.
- Shopping Cart Management: Users can add products to a cart, update item quantities, and remove items before proceeding to checkout. Cart contents are persisted.
- Secure Checkout with Paystack: Provide a secure checkout process where customers input delivery information and complete payment using Paystack's card and mobile money integrations.
- Customer Order History & Wishlist: Allow authenticated users to view their past orders and track their delivery status, as well as save products to a personal wishlist.
- Admin Dashboard for Content & Orders: A protected administrative interface for the brand to manage products (add, edit, delete with Firebase Storage for images) and view/update customer order statuses.
- AI Product Description Tool: Within the admin dashboard, a tool to generate concise, luxurious-sounding product descriptions based on provided attributes (e.g., material, color, style) to assist in content creation.

## Style Guidelines:

- Primary color: A deep, muted rose (#994D76) to convey elegance and femininity.
- Background color: A very pale pink, almost white (#FCF7F9), to maintain a clean, luxurious feel.
- Accent color: A vibrant fuchsia (#E680CC) for interactive elements and highlights, adding a modern touch.
- Additional accents: Gold for subtle luxury highlights, such as premium product badges or specific UI elements; Black for body text for clarity and contrast.
- Headline font: 'Playfair', a modern serif for an elegant and fashionable aesthetic. Body font: 'PT Sans', a humanist sans-serif, for professional and readable product presentations.
- Use clean, minimal, and sophisticated line icons that complement the premium and modern aesthetic of a luxury fashion brand.
- Implement a mobile-first responsive design with generous use of whitespace to create a luxury feel and structured content presentation, akin to fashion magazines.
- Subtle and smooth transition animations for product browsing and page navigation, enhancing the user experience without being distracting.
